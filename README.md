# Проект
Проект по созданию автономного IoT-узла для мониторинга окружающей среды с измерением температуры, давления и влажности. Данные выводятся в Serial-порт с возможностью последующей интеграции в IoT-платформы. Целевое применение: системы умного дома, метеостанции, промышленный мониторинг.

## Tech Stack
Язык: C++17 (с ограничениями для ESP8266)
Фреймворк: Arduino Framework (через PlatformIO)
Платформа: ESP8266 (Wemos D1 mini ESP-WROOM-02)
Сенсор: BME280 (I2C интерфейс)
Инструменты: PlatformIO Core, VSCode с расширениями
Система контроля версий: Git

### Architecture Patterns

Изоляция сенсорного кода через интерфейсы

Resource Management:

Explicit resource acquisition in setup()

## Domain Context
Физические величины:
Температура: градусы Цельсия, точность ±0.5°C
Давление: гектопаскали (hPa), абсолютное значение
Влажность: относительная влажность (%), диапазон 0-100%
Окружающая среда:
Рабочая температура: -20°C до +60°C
Влажность: 0-100% без конденсации
Высота над уровнем моря влияет на калибровку давления
Аппаратные особенности:
ESP-WROOM-02: 4MB flash, 80MHz тактовая
BME280: I2C адрес 0x76 или 0x77
Потребление: ~80mA в пике, <1mA в deep sleep

## Important Constraints
Технические:
Память:
Всего RAM: 80KB (система использует ~30KB)
Максимальный размер стека: 4KB
Избегать динамического выделения памяти
Время выполнения:
Loop должен выполняться <100ms
BME280 измерение: до 10ms
I2C timeout: 1000ms
Точность:
Температура: ±0.5°C после калибровки
Давление: ±1hPa
Влажность: ±3%
Бизнес-ограничения:
Стоимость компонентов: Минимизация дополнительных компонентов
Энергопотребление: Возможность работы от батареи 18650 (3.7V)
Надежность: MTBF >10,000 часов
Безопасность:
Over-the-Air обновления: Не реализовывать в первой версии
Защита от сбоев: Watchdog timer обязателен
Электробезопасность: Изоляция от высокого напряжения

## External Dependencies
Библиотеки (platformio.ini):
lib_deps = 
    adafruit/Adafruit BME280 Library@^2.2.2
    adafruit/Adafruit Unified Sensor@^1.1.6
    esp8266/ESP8266WiFi@^1.0  ; для будущего расширения
Аппаратные зависимости:
BME280:
Интерфейс: I2C (SDA=D2, SCL=D1 на Wemos D1 mini)
Напряжение: 3.3V
Pull-up резисторы: 4.7kΩ на SDA/SCL
Питание:
USB: 5V через Micro-USB
Батарея: через VIN (3.7-6V)
Serial для отладки:
USB-to-UART: CH340G
По умолчанию: 115200 baud
Инструменты разработки:
PlatformIO CLI: Версия >= 6.1
VSCode расширения:
PlatformIO IDE
C/C++ Extension Pack
Serial Monitor
Системные драйверы: CH340G для Windows/Mac
